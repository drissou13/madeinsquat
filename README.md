# madeinsquat
🏚️ Mad(e) in Squat — Le Flim Interactif
"Un patchwork décousu, un signal Wi-Fi d'outre-tombe et un hommage vibrant à la poésie marseillaise."

Ce projet est l'expérience web immersive du film Mad(e) in Squat (Kollectif Kokunka). Plus qu'un site vitrine, c'est une interface "hantée" qui réagit aux actions de l'utilisateur.

🎭 L'Expérience "Mister F"
Le site semble normal jusqu'à l'activation du bouton ⚠️ PANIQUE. Ce déclencheur lance une séquence scriptée :

Alerte Police : Gyrophares et sirènes (évacuation imminente).

Hantise (3 min) : Le site bascule dans la dimension de Mister F, le spectre pirate de la Plaine.

Audio : Le rire en écho de Mister F tourne en boucle (via Dropbox).

Visuel : Inversion spectrale des couleurs et instabilité des éléments (DOM possession).

Hommage : Injection forcée d'un texte dédié à Axel Toursky.

Shutdown : Extinction brutale type "Tube Cathodique" (CRT) à la fin du chrono.
